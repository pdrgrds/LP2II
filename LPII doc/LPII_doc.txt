-------------------------------USER-----------------------------------------
La [Content-Star] es un cinema, distribuido en varias zonas del país
destinado para la reprodución y administración de las peliculas.
Cuenta con todo tipo de servicios al consumo al cliente, tiene como objetivo
aumentar la agilidad de sus procesos de compra online.

El proceso empieza cuando el "Cliente" entra a la página web de [Content-Star]
despues de haber logueado (GUI), si no es asi, el cliente deberá registrarse(GUI),
ingresar los datos(Dni, Nombre, Apellido, Nro. Telefonico, Dirección).

El cliente visualizará las pantalla de Menú(GUI), donde se mostraá las promociones
en comida y las peliculas.

El cliente selecciona una pelicula, si se registro como tipo de usuario PREMIUN, 
obtendrá Promociones y preferencias en la pelicula que escoja. Si no es asi, 
el cliente deberá escoger una fecha disponible dada por el sistema.

El sistema mostrará si una opcion si está conforme con su pedido. Si es asi,
el cliente realiza la compra.

El sistema genera el comprobante de pago y finaliza el proceso.
